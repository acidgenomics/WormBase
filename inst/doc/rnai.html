RNAi clone mapping

Michael J. Steinbaugh

2016-10-03

RNA interference libraries are commonly used in C. elegans research. RNAi clones are typically referenced by a gene pair or open reading frame (ORF) sequence, which are often difficult to map to an up-to-date gene identifier. We have written the rnai() function to solve this problem. The function enables a researcher to search for RNAi clones by a gene identifier, or map an RNAi clone by plate location and return the up-to-date WormBase gene identifier. The function also obtains primary and secondary target information from WormBase, which can be informative since many RNAi clones have off-target effects.

The function has built-in support for the commonly used Ahringer and ORFeome v1.1 RNAi libraries, as well as the cherrypick libraries designed by the Ruvkun Lab.

Alternatively, you can search by WormBase gene identifier or sequence, aka open reading frame (ORF). These queries will return the same result:

rnai("WBGene00004735", format = "gene")
rnai("Y47D3B.7", format = "sequence")

Mapping RNAi clones to gene identifiers

The rnai() function was written to simplify the task of identifying wells of interest in an RNAi screen. You can simply input the well identifiers by plate location, and rnai() will return up-to-date gene identifiers from WormBase.

Ahringer library

Hereâ€™s how you map wells from the Ahringer library in 384 well format. Note that plate location identifiers must include the chromosome in roman numeral format at the beginning:

t(
    rnai("III-6C01", library = "ahringer384")
)
gene WBGene00004735
historical JA:Y47D3B.7
length 932
name sbp-1
oligo sjj_Y47D3B.7
rnai WBRNAi00009345
sequence Y47D3B.7
targetPrimary WBGene00004735
targetSecondary NA
clone III-6C01

Wells from the Ahringer library in 96 well format are also supported. Since this library has unique plate numbers, the chromosome prefix is not necessary:

t(
    rnai("86B01", library = "ahringer96")
)
gene WBGene00004735
historical JA:Y47D3B.7
length 932
name sbp-1
oligo sjj_Y47D3B.7
rnai WBRNAi00009345
sequence Y47D3B.7
targetPrimary WBGene00004735
targetSecondary NA
clone 86B01

ORFeome library

rnai() has mapping support for the ORFeome library, in 96 well format. Plate position should be entered in the same format as the WORFDB. The function is flexible, and also supports entries in the historical format (e.g. GHR-11010@G06), or without a separator:

t(
    rnai("11010G06", library = "orfeome96")
)
gene WBGene00004735
historical MV_SV:mv_Y47D3B.7
length 14145
name sbp-1
oligo mv_Y47D3B.7
rnai WBRNAi00037170
sequence Y47D3B.7
targetPrimary WBGene00004735
targetSecondary NA
clone 11010G06

Cherrypick libraries

rnai() supports clone mapping for cherrypick libraries from the Ruvkun Lab, including the G protein coupled receptor (GPCR), kinase, nuclear hormone receptor, and transcription factor libraries. These identifiers require the following prefixes:

  • gpcr
  • kinase
  • nhr
  • tf_all
  • tf_bzip

Hereâ€™s how you query well E01 (daf-16) from the first plate of the transcription factor cherrypick library:

)
